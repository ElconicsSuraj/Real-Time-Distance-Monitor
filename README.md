# Real-Time-Distance-Monitor
Real-Time Distance Monitoring System using ESP32 and TF-Luna LiDAR

Developed an embedded system that measures distance using the TF-Luna LiDAR sensor interfaced with an ESP32 microcontroller. The ESP32 hosts a lightweight web server that allows users to monitor distance measurements in real time through any web browser connected to the same network. The system continuously reads sensor data via UART communication and updates the distance values on the web interface with minimal latency.

Key Features

Real-time distance measurement using TF-Luna LiDAR sensor

ESP32-based web server for wireless monitoring

Live distance updates accessible through any browser

Efficient UART communication for sensor data acquisition

Lightweight and responsive web interface

Technologies Used

ESP32

TF-Luna LiDAR Sensor

Embedded C / Arduino Framework

UART Communication
